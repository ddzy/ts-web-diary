# ts-web-diary

## 说明

此为`2018/9/2` - `2019/1/18`完成的第一版.

为了后期的版本参考, 故将其归纳为`branch/rel-1`分支💥

由于初学`react`, 所以只完成了`小学期`制定的基本任务:

- [x] jwt权限认证
- [x] 登陆注册
- [x] 首页文章分类
- [x] 文章发布、编辑、详情、点赞、收藏
- [x] 个人中心
  - [x] 个人文章分类
  - [x] 个人文章编辑
  - [x] 个人收藏夹编辑
  - [x] 个人信息更改
- [x] 页面路由过渡

## 依赖

- Node >= v10.3.0
- MongoDB >= v3.6.4

## 启动

### Clone

```bash
git clone https://github.com/ddzy/ts-web-diary.git
```

### Server

```bash
npm run server
```

### Client

```bash
npm run start
```

**Enjoy!**
